# Electron-TypeScript-React

### Create An Electron App with TypeScript, React, Redux, React-Router

## Installation
```bash
$ npm install -g electron-typescript-react
```

## Usage
```bash
$ build-app init <project-name>
```

#### Example
```bash
$ build-app init electron-project
```
